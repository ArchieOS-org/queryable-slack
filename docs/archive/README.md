# Documentation Archive

This directory contains historical documentation, debug reports, and deployment logs from the development process.

## Structure

- **`debug-reports/`**: Debug sessions, error investigations, and troubleshooting documents
- **`deployment-logs/`**: Deployment attempts, configuration changes, and setup guides

## Purpose

These files are archived for historical reference and to preserve the development journey. They document:

- Issues encountered during development
- Solutions and workarounds discovered
- Deployment configurations that were tested
- Integration attempts and results

## Note

For current, up-to-date documentation, refer to the main `docs/` directory:
- `docs/README.md` - Project overview
- `docs/SETUP.md` - Setup instructions
- `docs/DEPLOYMENT.md` - Deployment guide
- `docs/ARCHITECTURE.md` - System architecture
- `docs/API.md` - API documentation
