# Setup Credentials - Step by Step

## What I Need From You

### Step 1: Supabase Organization ID

1. Go to https://supabase.com/dashboard
2. Click on your organization (or create one if you don't have one)
3. Copy the **Organization ID** from the URL or settings
   - It looks like: `cool-green-pqdr0qc` or similar

**I need:** Your Supabase Organization ID

---

### Step 2: Create Supabase Project

Once I have your org ID, I'll help you create the project. You'll need to provide:
- A secure database password (I'll generate one or you can provide)

---

### Step 3: After Project Creation

After the project is created, I'll need:
- **Project Reference ID** (from the project URL or dashboard)
- **SUPABASE_URL** (from Project Settings → API)
- **SUPABASE_ANON_KEY** (from Project Settings → API → anon public key)
- **SUPABASE_SERVICE_ROLE_KEY** (from Project Settings → API → service_role key - keep secret!)
- **DATABASE_URL** (from Project Settings → Database → Connection string)

---

### Step 4: Vercel Setup

I'll help you:
- Link the project to Vercel
- Set up environment variables
- Deploy

---
