# How Agents Normally Say They Have Listings

Based on analysis of the queryable-slack database using the agent listing prompt, here are the common patterns for how real estate agents announce listings:

## Common Greeting Patterns

### Casual/Friendly Greetings
- **"Hi Everyone"** - Most common for team announcements
- **"Hi guys"** - Informal team communication
- **"Hello Everyone"** - Slightly more formal but still friendly

### Professional/Internal Greetings
- **"Hello <@mention>"** - Used for direct team member communication
- Includes Slack user mentions for specific coordination

## Introduction Phrases

### Direct Ownership Statements
Agents typically use first-person ownership phrases:

1. **"I have an [listing type]..."**
   - Example: "I have an Assignment Sale in Muskoka Bracebridge Ontario"
   - Example: "I have buyers with a budget of up to $1.1M..."

2. **"I also have an [listing type]..."**
   - Example: "I also have an exclusive lease listing at Yorkville Ave"
   - Example: "I also have buyers with a budget..."

3. **Opportunity-focused introductions**
   - Example: "INCREDIBLE INVESTMENT OPPORTUNITY IN USA residential properties"

### Status Update Phrases (Internal)
- **"the photos are ready for [address]"**
- **"Pls see the MLS draft"**
- **"I will work on the listing"**

## Property Detail Presentation

### Structure Pattern
Agents typically present details in this order:
1. **Location/Address** - Always mentioned early
2. **Property Type** - Bedrooms, bathrooms, square footage
3. **Price/Rental Rate** - Clear pricing information
4. **Special Features** - Lot size, upgrades, amenities
5. **Status** - Availability, closing dates, deposit info
6. **Context** - Neighborhood, subdivision, positioning

### Presentation Styles

**Bullet-Point Style (Most Common):**
```
4983 square feet
4 bdrms
4.5 bathrooms 
Huge Ravine lot #76
Purchase price $2,197,000
Deposit paid 10%
Asking $2,697,000 +HST
```

**Narrative Style:**
```
"I also have an exclusive lease listing at Yorkville Ave. 1+1 and 1 Bath.
2899/month or Fully furnished for 3599/month. It is vacant. Ready to show at anytime."
```

## Pricing Communication Formats

### Sale Listings
- **"Asking $[amount] +HST"**
- **"Purchase price $[amount]"**
- **"Deposit paid [percentage]%"**
- **"Asking $[amount]"**

### Lease Listings
- **"[amount]/month"**
- **"[amount]/month or [option] for [amount]/month"**
- Example: "2899/month or Fully furnished for 3599/month"

## Call-to-Action Phrases

### Collaborative Requests
- **"If you have any potential renters, please help me lease this unit!"**
- **"please help me lease this unit!"**

### Professional Invitations
- **"please reach out to me for more information"**
- **"Pls let us know in this slack"**

### Approval Requests (Internal)
- **"Please review the Description: Once Approved, pls slack and let us know and write Approved"**

### Listing Type Descriptors

## Tone Variations

### Casual Collaborative (Team Sharing)
- Uses "Hi Everyone"
- Enthusiastic language
- Collaborative requests ("please help me")
- Exclamation marks
- Direct listing URLs

### Professional Detailed (High-Value Listings)
- Uses "Hello Everyone"
- Structured, comprehensive details
- Professional invitation ("reach out to me")
- Bullet-point format
- Context and positioning included

### Internal Coordination (Team Workflow)
- Uses "Hello <@mention>"
- Task-oriented language
- Approval-focused
- Status updates
- Direct instructions

## Key Takeaways

1. **Always start with a greeting** - "Hi Everyone" or "Hello Everyone"
2. **Use first-person ownership** - "I have" or "I also have"
3. **Specify listing type** - "exclusive lease listing", "Assignment Sale", etc.
4. **Include location early** - Address or area mentioned in first sentence
5. **Present details clearly** - Either narrative or bullet-point style
6. **Always include pricing** - Clear, with currency symbols
7. **End with call-to-action** - Request for help, contact, or approval
8. **Add context when valuable** - Neighborhood, subdivision, special features

## Variations by Context

- **Team Channels (buyer-needs-and-wants)**: More casual, collaborative, includes links
- **Internal Channels (agent-specific)**: More formal, task-oriented, approval-focused
- **High-Value Listings**: More detailed, professional tone, comprehensive specs
- **Lease Listings**: More casual, urgent availability language, collaborative requests
